Welcome to dbconsole

Use this console to link to the databases of all your Salesforce.com, Force.com and Database.com accounts for easy web login and access via the dbc command line tool.

  1. Download the dbc command line tool with 
    $ curl http://dbc-api.herokuapp.com/tool/dbc > $HOME/bin/dbc
$ chmod +x $HOME/bin/dbc
  2. Log in to this web console with any Salesforce.com account. The account will become the "master" account for managing other accounts.
  3. Go get your API key under settings and pass it to the command line tool with 
    $ dbc key <apikey>
  4. Use the web console to set up links to all your Salesforce.com, Force.com or Database.com developer accounts.
